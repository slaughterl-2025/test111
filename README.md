# 基于聚类分层与关联规则的湖南省招聘平台差异化策略研究

> K-Means 聚类 + Apriori 关联规则 · 招聘市场岗位特征挖掘

## 项目详细介绍

### 1. 项目背景与数据概况

本项目核心数据来源于 **智联招聘平台**，采集了湖南省及周边地区近 **9.3 万条**真实的在线招聘岗位记录，经过数据清洗后得到有效记录约 **8.1 万条**。每条记录包含 20 个字段，涵盖公司名称、招聘岗位、所在城市、工作区域、最低/最高月薪、学历要求、经验要求、招聘类型、所属行业、来源平台等维度。

项目要回答四个核心问题：第一，不同城市、不同行业的高薪岗位有什么特征？第二，哪些条件组合意味着某个岗位大概率是高薪？第三，招聘市场上岗位可以分成几个层次？第四，基于分析结果能为平台或求职者提供什么策略建议？

### 2. 技术路线：双分支并行分析

**数据预处理分支**共八个步骤：删除无效记录、计算平均月薪、100+ 细分行业归并为 9 大行业类、学历 1~6 级量化编码、经验 0~12 数值编码、城市 Top 7 归类、薪资等频分箱四档（低薪/中薪/高薪/超高薪）。

**K-Means 聚类分支**以经验值、学历等级、平均月薪为变量，经 Z-Score 标准化后，结合肘部法则与轮廓系数确定最优 **K = 4**，将岗位分为四层：中级技术岗（65.1%，平均月薪 ¥9,531）、初级基础岗（18.8%，¥7,183）、资深专业岗（15.9%，¥16,584）、高端管理/技术岗（0.2%，¥140,360）。

**Apriori 关联规则分支**将平台、城市、行业、薪资分箱、岗位层级五维离散变量独热编码为 26 列 01 矩阵，设定最小支持度 2%、最小置信度 50%、提升度 > 1，共挖掘 **270 条有效关联规则**。

### 3. 核心发现

- **规则一**：{长沙, 资深专业岗, 其他行业} → {超高薪}，Lift = 3.05，置信度 62%。即长沙的经验丰富人才即使不在热门行业，仍有 3 倍概率拿超高薪
- **规则二**：{IT/互联网} → {长沙, 中级技术岗}，Lift = 1.72，置信度 60.5%，反映湖南 IT 产业高度集中于长沙
- **规则三**：{初级基础岗, 其他行业} → {低薪}，Lift = 1.96，入门级岗位薪酬普遍偏低
- **行业薪资**：IT/互联网（¥12,465）> 能源/环保（¥11,128）> 制造业（¥11,081），长沙占据 66% 的岗位量

### 4. 可视化与策略建议

项目生成 8 张图表：肘部轮廓联合判别图、聚类散点图、城市×行业热力图、行业×岗位层级堆叠柱状图、城市×岗位层级分布图、薪资箱线图、关联规则散点图及网络图。

策略建议：智联招聘在湖南省中级技术岗、IT 互联网和制造业领域已形成护城河；拓展方向可加大高端管理岗和资深专业岗的推荐算法倾斜，同时开辟株洲、湘潭等二三线城市的 IT 岗位蓝海市场。求职者若追求高薪，进入 IT/互联网或能源环保行业、选择长沙、积累七年以上经验是最有效路径。

### 5. 技术栈

Pandas + NumPy 数据处理，Scikit-learn 聚类与标准化，Mlxtend Apriori 关联规则，Matplotlib + Seaborn + NetworkX 可视化。全程 Python 实现，代码约 300 行，运行约 2 分钟。

## 项目概述

本项目基于 **智联招聘平台 9.3 万条**湖南省真实招聘数据，综合运用 **K-Means 聚类**与 **Apriori 关联规则算法**，从"平台 × 城市 × 行业 × 薪资 × 岗位层级"五个维度挖掘岗位特征的多维关联模式，为平台运营与求职者策略提供数据驱动的差异化建议。

## 数据预处理

1. **缺失值处理** — 删除月薪为空/为 0、招聘类型为 NaN 的无效行
2. **薪资字段** — 新增 `平均月薪 = (最低月薪 + 最高月薪) / 2`
3. **行业分类** — 100+ 细分行业映射为 9 大行业类
4. **学历量化** — 从"初中"到"博士"映射为 1~6 数字等级
5. **经验量化** — 从"不限"到"10年以上"映射为 0~12 数值
6. **城市归类** — 保留 Top 7 城市，其余归为"其他城市"
7. **薪资分箱** — 等频分箱 4 档：低薪 / 中薪 / 高薪 / 超高薪
8. **平台处理** — 平台字段标准化

## K-Means 聚类结果

以 **经验值、学历等级、平均月薪** 三个变量聚类，肘部法则 + 轮廓系数确定 **最优 K = 4**（轮廓系数 0.465）：

| 0 | 中级技术岗 | 1.9 年 | 本科 | ¥9,531 | 65.1% |

| 2 | 资深专业岗 | 7.2 年 | 本科 | ¥16,584 | 15.9% |
| 3 | 高端管理/技术岗 | 3.3 年 | 硕士 | ¥140,360 | 0.2% |

## 核心结论

1. **长沙是绝对中心** — 66% 的岗位集中在长沙，IT/互联网行业高度集中于省会
2. **资深人才薪资溢价显著** — 7 年以上经验 + 长沙 = 超高薪概率提升 3 倍 (Lift=3.02)
3. **IT/互联网薪资领跑** — 平均月薪 12,465 元，居行业首位
4. **服务业体量最大但薪酬偏低** — 占 82% 的岗位量，薪资中位数较低
5. **平台策略建议** — 智联招聘在湖南省中级技术岗领域已形成护城河，可向高端岗和二三线城市拓展
